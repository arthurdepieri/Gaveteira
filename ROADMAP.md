# Roadmap da Gaveteira

Lista curta de próximos passos para manter o projeto organizado entre versões beta.

1. [ ] Recuperação de senha.
2. [ ] Trocar `localStorage` por IndexedDB quando o volume de imagens/anexos crescer.
3. [ ] Ampliar clientes reais de metadados em `src/services/metadata.ts`.
4. [ ] Melhorar upload local de capas e fotos, além de URLs.
5. [ ] Criar modo de recomendação para decidir o próximo item com base em nota, gênero, status e tempo disponível.
6. [ ] Revisar fluxo premium/familiar antes de qualquer monetização.
7. [ ] Planejar o jogo de cartas da Gaveteira como experimento pós-lançamento.
