# Elementos sazonais

Esta pasta guarda os elementos visuais das fichas de temporada.

Use `promocoes/<id-da-promocao>/` para manter cada temporada separada. O `id-da-promocao` deve combinar com o `id` cadastrado em `src/data/seasonalThemes.ts`.

Exemplos de arquivos que podem morar aqui depois:

- imagens decorativas
- texturas
- padroes de fundo
- selos
- recortes transparentes
- referencias visuais aprovadas
