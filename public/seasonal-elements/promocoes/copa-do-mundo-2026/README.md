# Copa do Mundo 2026

Promocao ativa para fichas criadas de 2026-06-14 ate 2026-07-20.

O id usado no app e `copa-do-mundo-2026`.

Coloque arquivos finais em:

```text
public/seasonal-elements/promocoes/copa-do-mundo-2026/elementos/
```

No app, eles podem ser referenciados com:

```text
/seasonal-elements/promocoes/copa-do-mundo-2026/elementos/nome-do-arquivo.svg
```
